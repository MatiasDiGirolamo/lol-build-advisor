Draft Companion

Privacy Policy

Last updated: March 28, 2026.

Overview

Draft Companion provides League of Legends profile, champion build, and live game companion features. We only request the minimum player information needed to show Riot account profile data, ranked queues, recent matches, champion statistics, and live game recommendations.

Data We Process

When you search for a Riot ID, the app may process your Riot Game Name, Tag Line, platform/region, profile icon, summoner level, ranked queue data, champion mastery data, recent match data, and active-game spectator data when available through Riot APIs.

How We Use Data

We use Riot account and gameplay data solely to render the player hub, champion pages, recent performance summaries, and live game adaptation recommendations. We do not sell personal data and we do not use Riot account data for unrelated advertising.

Stored Data

The app stores recent searches and favorite Riot IDs locally in your browser to improve autocomplete and usability. These browser-side preferences remain on your device unless you clear your local storage.

Third-Party Services

This product uses Riot Games APIs, Riot Data Dragon assets, and public meta sources. Their own terms, privacy policies, and availability rules may apply independently.

Security

We apply basic transport and API safeguards, including server-side rate limiting, security headers, and secret handling through environment variables. No security system is perfect, but we take reasonable steps to reduce unnecessary exposure.

Contact

If you need to request deletion of browser-side preferences or have privacy questions about this project, contact the operator of this site using the contact information published alongside the product.
